Instruksi Pengerjaan Tugas

Silahkan Baca Instruksi Dibawah

1. Klik Tombol Bertuliskan “Ngerjain Kuy” pada tugas yang ingin anda kerjakan ataupun lihat

2. Anda akan masuk kedalam Google Document, didalamnya ada Soal dan Jawaban

3. Jika Anda Lihat di Jawaban ada x (Admin), itu adalah tempat Admin Mengisi Jawaban untuk setiap nomornya, Mohon Jangan Mengisi Jawaban Disitu

4. Tapi Isilah Jawaban pada Kolom “Jawaban Lain” yang telah disediakan, dan jangan lupa untuk menuliskan nama kalian setelah mengerjakan di akhir jawaban kalian

5. Setelah memeriksa Jawaban Kalian Admin akan Memberikan Poin (Fungsi Poin dijelaskan di bawah)

6. Jangan Takut untuk memberikan Jawaban Kalian walaupun Admin blm memberikan Jawaban, Dan Selalu ingat untuk jangan mengerjakan Jawaban di tempat Admin ,walau kalian sudah lebih dulu dari Admin
